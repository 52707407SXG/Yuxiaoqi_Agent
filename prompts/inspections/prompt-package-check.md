# Inspection Prompt: PromptPackage Check

检查 PromptPackage 是否合格：

- 是否明确作品类型、平台、比例、时长或篇幅。
- 是否说明受众、目标、素材来源和禁用项。
- 是否包含脚本或正文结构、分镜或版式、画面/声音/字幕提示词。
- 是否列出 ToolPlan、预计成本范围、耗时、确认需求和失败处理。
- 是否只引用授权 SourceBrief 和 asset ref。
- 是否避免密钥、cookie、token、内部路径、客户隐私和跨账号资料。
- 是否能被 My Stand 后端审计和复现。

若不合格，输出失败原因和最小返工清单。
